### Try It Now

```json
{
  "mcpServers": {
    "antigravity-cloud": {
      "url": "https://openclaw-antigravity-cloudserver.yagami8095.workers.dev/mcp"
    }
  }
}
```

Add this to your `.mcp.json` — 8 AI tools, zero cost, zero setup.
